# Fitness Membership Analytics Dashboard

## Overview
This project analyzes a fitness membership dataset using Microsoft Excel. It covers the complete data analytics workflow including data cleaning, modeling, KPI development, customer segmentation, forecasting, and dashboard creation to generate actionable business insights.

---

## Key Features
- Data cleaning and preprocessing  
- KPI Engine (Total Revenue, Active Members, Churn Rate, ARPU, Engagement)  
- Customer Segmentation using RFM Analysis  
- Cohort Analysis for retention tracking  
- Forecasting using moving averages and linear regression  
- Interactive dashboard with slicers  
- Scenario analysis for revenue projection  
- Automated insights generation  

---

## Dashboard Highlights
- Revenue trend and forecast visualization  
- City-wise (regional) performance analysis  
- Membership type performance comparison  
- Customer segmentation insights  

---

## Key Insights
- Revenue shows fluctuations with no consistent upward trend  
- A significant portion of customers are at risk, impacting revenue stability  
- Standard membership drives the highest revenue and value  
- Revenue is concentrated in a few high-performing cities  
- Forecast indicates stable to slightly declining future performance
